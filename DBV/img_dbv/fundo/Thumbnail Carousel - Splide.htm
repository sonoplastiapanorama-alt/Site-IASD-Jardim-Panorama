Introduo

Um carrossel pode sincronizar com outro carrossel por sync()mtodo, e isNaviationopo torna o carrossel clicvel para navegao. Usando esses 2 recursos, podemos fazer um carrossel com um controle de miniatura como este:

Esta pgina explica como criar esse tipo de carrossel.

O carrossel de miniaturas tem que exibir vrios slides em uma pgina. Normalmente, ns usamos o perPageopo para definir o nmero de slides, mas difcil determinar o nmero da perspectiva da responsabilidade mesmo que usando pontos de interrupo.

Para resolver este problema, vamos usar fixedWidthpara fixar a largura do slide contra o tamanho da janela.

document.addEventListener( 'DOMContentLoaded', function () {
  new Splide( '#thumbnail-carousel', {
		fixedWidth: 100,
		gap       : 10,
		rewind    : true,
		pagination: false,
  } ).mount();
} );
JavaScript

		pagination: false,
  } ).mount();
} );

E ento, precisamos encaixar a miniatura na dimenso do slide. Se todas as miniaturas tiverem a mesma proporo, o CSS a seguir suficiente:

.splide__slide img {
  width: 100%;
  height: auto;
}
CSS
.splide__slide img {
  width: 100%;
  height: auto;
}

Se no, vamos usar object-fitpara cobrir cada slide com uma miniatura (o IE no suporta esta propriedade!). Para que funcione, precisamos definir fixedHeight:

document.addEventListener( 'DOMContentLoaded', function () {
  new Splide( '#thumbnail-carousel', {
		fixedWidth : 100,
    fixedHeight: 60,
		gap        : 10,
		rewind     : true,
		pagination : false,
  } ).mount();
} );

    fixedHeight: 60,

Parece muito bom, mas h algumas coisas que devemos considerar:

  • O usurio no pode informar qual slide selecionado
  • 100px provavelmente grande para dispositivos mveis
  • O slide ativo deve estar no centro, se possvel

Se voc estiver usando o tema CSS, como splide.min.cssou splide-skyblue.min.css, enfatiza o slide ativo adicionando um estilo de borda. Caso contrrio, voc tem que estiliz-lo manualmente pelo is-activeclasse de status:

.splide__slide {
  opacity: 0.6;
}


.splide__slide.is-active {
  opacity: 1;
}
CSS
.splide__slide {
  opacity: 0.6;
}


.splide__slide.is-active {
  opacity: 1;
}

O breakpointse focusopes nos ajudam a lidar com o segundo e terceiro problemas. Se voc no estiver usando object-fit, fixedHeightno necessrio.

new Splide( '#thumbnail-carousel', {
  fixedWidth : 100,
  fixedHeight: 60,
  gap        : 10,
  rewind     : true,
  pagination : false,
  focus      : 'center',
  breakpoints: {
    600: {
      fixedWidth : 60,
      fixedHeight: 44,
    },
  },
} ).mount();

Carrossel Principal

Este tutorial usa o seguinte carrossel de desbotamento. Voc tem que atribuir uma identificao diferente, como main-carousel, ou uma classe para agarr-lo mais tarde por script.

Este exemplo desativa flechas e paginao, mas se voc quiser habilit-los, basta excluir essas opes.

Sincronizao

Para sincronizar o carrossel principal com miniaturas, use o Splide#sync()mtodo. Voc deve cham-lo antes mount().

var main       = new Splide( '#main-carousel', { ... } );
var thumbnails = new Splide( '#thumbnail-carousel', { ... } );


main.sync( thumbnails );
main.mount();
thumbnails.mount();

main.mount();
thumbnails.mount();
